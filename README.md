PyPlatformerEngine 
==================

#### Description
Python-based Platformer Base Engine and utilities for developing simple 2d platformers. Using as many professional patterns as possible.

#### Features
* Physics
	* Gravity
* Objects
	* Sprite management
	* Character management
	* Object management
	* Level management
